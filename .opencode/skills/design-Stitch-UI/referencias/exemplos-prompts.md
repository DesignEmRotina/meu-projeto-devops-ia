## Painéis

### Painel de Análise

```
Painel de análise da web para a equipe de marketing

Barra Superior:
- Período Seletor (últimos 7 dias, 30 dias, personalizado)
- Botão de exportação
- Ícone de notificação (tipo sino)
- Menu suspenso de perfil do usuário

Linha de métricas principais:
- 4 cartões de métricas em uma linha
- Cartão 1: Total de visitantes (com variação percentual)
- Cartão 2: Taxa de conversão (com gráfico de linhas de tendência)
- Cartão 3: Taxa de rejeição (com comparação ao período anterior)
- Cartão 4: Duração média da sessão

Gráfico principal:
- Gráfico de linhas mostrando o tráfego ao longo do tempo
- Várias linhas para diferentes fontes (Orgânica, Paga, Social, Direta)
- Legenda interativa para alternar as linhas
- Dicas de ferramentas ao passar o mouse com valores exatos

Painéis secundários:
- Esquerda: Tabela de páginas principais (página, visualizações, tempo médio, taxa de rejeição)
- Direita: Gráfico de pizza de fontes de tráfego com porcentagens

Seção inferior:
- Tabela de conversões recentes com usuário, fonte, valor e data/hora

Estilo: Limpo, focado em dados, profissional
Cores: Barra lateral azul-marinho, área principal branca, gráfico colorido linhas
Tipografia: Monoespaçada para números, sem serifa para rótulos
Plataforma: Web para desktop (1440px+)
```

## Dicas para usar estes exemplos

1. **Personalize para suas necessidades** - Substitua o conteúdo de exemplo por seus requisitos específicos
2. **Combine elementos** - Misture e combine componentes de diferentes exemplos
3. **Ajuste a complexidade** - Simplifique ou expanda com base no escopo do seu projeto
4. **Especifique **Sua marca** - Adicione sua paleta de cores, fontes e estilo visual
5. **Considere a plataforma** - Adapte os layouts para o seu dispositivo alvo (celular/desktop)
6. **Adicione contexto** - Inclua personas de usuário ou casos de uso para obter melhores resultados
7. **Iterar** - Comece com uma sugestão básica e refine com anotações

Lembre-se: estes são pontos de partida. O Stitch funciona melhor quando você fornece detalhes específicos relevantes para o seu projeto.
