# Padrões de Teste do Bats

Guia completo para escrever testes unitários abrangentes para scripts shell usando o Bats (Sistema de Teste Automatizado do Bash), incluindo padrões de teste, fixtures e melhores práticas para testes de shell em nível de produção.

## Use esta habilidade quando:

- Escrever testes unitários para scripts shell
- Implementar TDD para scripts
- Configurar testes automatizados em pipelines de CI/CD

- Testar casos extremos e condições de erro

- Validar o comportamento em diferentes ambientes shell

## Não use esta habilidade quando:

- O projeto não usa scripts shell
- Você precisa de testes de integração além do comportamento do shell
- O objetivo é apenas linting ou formatação

## Instruções

- Confirme os dialetos shell e os ambientes suportados.

- Configure uma estrutura de testes com helpers e fixtures.

- Escreva testes para códigos de saída, saídas e efeitos colaterais.

- Adicione testes de setup/teardown e execução em CI.

- Se precisar de exemplos detalhados, abra `recursos/manual-de-implementação.md`.
