# Padrões GDScript do Godot

Padrões de produção para desenvolvimento de jogos com GDScript no Godot 4.x, abrangendo arquitetura, sinais, cenas e otimização.

## Use esta habilidade quando

- Criar jogos com Godot 4
- Implementar sistemas de jogos em GDScript
- Projetar a arquitetura de cenas
- Gerenciar o estado do jogo
- Otimizar o desempenho do GDScript
- Aprender as melhores práticas do Godot

## Não use esta habilidade quando

- A tarefa não estiver relacionada aos padrões GDScript do Godot
- Você precisar de um domínio ou ferramenta diferente fora deste escopo

## Instruções

- Esclareça os objetivos, restrições e entradas necessárias.

- Aplique as melhores práticas relevantes e valide os resultados.

- Forneça etapas práticas e verificações.

- Se forem necessários exemplos detalhados, abra o arquivo `recursos/manual-de-implementação.md`.
