Angular

Um guia completo para o desenvolvimento moderno em Angular (v20+) otimizado para agentes de IA e LLMs.

Visão geral

Esta habilidade abrange padrões modernos do Angular, incluindo:

Sinais - o recurso primitivo reativo do Angular para gerenciamento de estado

Componentes independentes - arquitetura de componentes moderna sem NgModules

Aplicações sem zonas - aplicativos de alto desempenho sem Zone.js

SSR e hidratação - padrões de renderização no servidor e hidratação no cliente

Roteamento moderno - guardas funcionais, resolvers e carregamento lento

Injeção de dependência - Injeção de dependência moderna com a função inject()

Formulários reativos - manipulação de formulários com segurança de tipos

Estrutura

Esta habilidade consiste em um único arquivo SKILL.md abrangente contendo:

Padrões de componentes modernos com entradas/saídas de sinais

Gerenciamento de estado com sinais e valores computados

Técnicas de otimização de desempenho

Melhores práticas de SSR e hidratação

Estratégias de migração de padrões legados do Angular

Uso

Esta habilidade foi projetada para ser lida na íntegra para compreender a abordagem completa do desenvolvimento moderno em Angular, ou consultada para padrões específicos quando necessário. Versão

Versão atual: 1.0.0 (fevereiro de 2026)
